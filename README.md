Please the folow the steps to clone the project and run the app

Step 1: Clone the project

```bash
$ git clone https://github.com/ShivashishY/chatter.git
```

Step 2: Packge installation

``` bash
$ npm install
```
Step 3: Run the project

``` bash
node server
```
